# GIF Forge

GIF Forge is a web application for creating animated GIFs from images and videos.

## Features

### Current Features
- Upload multiple images
- Generate animated GIFs
- Preview GIFs
- Download generated GIFs

### Planned Features
- Drag-and-drop image ordering
- Frame speed control
- Video to GIF conversion
- GIF compression
- Text overlays
- Cloud storage
- User authentication

## Tech Stack

### Phase 1
- HTML
- CSS
- JavaScript
- GIF.js

### Phase 2
- React
- Tailwind CSS

### Phase 3
- Node.js
- Express
- FFmpeg
